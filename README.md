## 租房注意事项

1. 租房合同类型应当是租赁合同，而非代理合同。
2. 转租不受法律保护，注意看民法条文。
3. 如果您是受害者建议通过报警和起诉的有效方式维护自己的合法利益。法律防君子不防小人，尽量收集证据。
4. 报警回执制度：群众到公安机关报警时，公安民警必须热情接待，认真登记并发给报警回执。对属于权限范围内的案件，一般要在一周内办妥，未能办妥的要说明原因；对不属于本辖区或本警种管辖的案件，要在两天内转交有关单位处理并向群众解释清楚。
